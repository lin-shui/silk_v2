# Context Index

本目录给 coding agent 使用，不是面向人的教程。

## Read Order

1. `AGENTS.md`
2. `ARCHITECTURE.md`
3. `TASK_ROUTER.md`
4. 任务对应的最小子文档

## Layers

- `generated/`: 仓库地图与检索入口
- `project/`: 构建、运行、存储、已知偏差
- `modules/`: backend / frontend 深挖
- `integrations/`: Claude Code、Weaviate、飞书等辅助子系统
- `quality/`: 测试面、快检、验证映射
- `planning/`: Todo 治理与执行计划入口
