# Frontend Index

## Frontend Topology

- `frontend/shared`: Web / Android / Desktop 共享合同与 WebSocket 客户端
- `frontend/webApp`: Compose for Web，当前功能最完整的浏览器端
- `frontend/androidApp`: Jetpack Compose，移动端主壳
- `frontend/desktopApp`: Compose Desktop，可编译/可测试，但当前 feature parity 较低
- `frontend/harmonyApp`: ArkTS/ArkUI，独立实现，不复用 `shared`
